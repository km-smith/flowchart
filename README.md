# Flowchart

## Getting started

1. Create a virtual environment
```zsh
python3 -m venv venv
```

2. Activate your virtual environment
```zsh
activate venv/bin/activate
```

3. Install deps
```zsh
pip install -r requirements.txt
```

4. Run the script (from the project root)
```zsh
python ./flowchart.py
```
